# 📖 Diary

這是一個屬於 **Nasir** 的專屬樹洞，記錄每一天的點滴與成長。

## 用意

這個 Repo 主要是要來豐富每一天，以及放置一些小小的 Side Project。

讓整個 Repo 確實記錄每一天的人生，像是一本專屬的日記本一樣。

## 內容

- **Blog 文章** — 日常紀錄、技術筆記、生活隨想
- **小 Project** — 一些有趣的 Side Project 作品
- **學習分享** — 學習過程中的知識沉澱
- **個人網頁** — 關於我自己的介紹

## 技術棧

- [Astro](https://astro.build/) — 靜態網站生成框架
- Google Fonts — 思源宋體 + 芫荽體
- GitHub Actions — 自動部署至 GitHub Pages

## 部署

推送到 main 分支後，GitHub Actions 會自動構建並部署至 GitHub Pages。
